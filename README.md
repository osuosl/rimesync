# rimesync

A Ruby Gem equivalent in functionality to [Pymesync](https://github.com/osuosl/pymesync).

## Build Instructions

```shell
$ git clone https://github.com/osuosl/rimesync
$ cd rimesync
$ bundle install
$ gem build rimesync.gemspec
$ gem install rimesync-0.1.0.gem
```

### Build docs

```shell
$ rdoc lib
```

### Running Tests

```shell
$ rake test
```
