    rimesync¶ ↑

    A Ruby Gem equivalent in functionality to Pymesync.

    Build Instructions¶ ↑

    $ git clone https://github.com/osuosl/rimesync
$ cd rimesync
$ bundle install

    Running Tests¶ ↑

    $ ruby tests.rb
$ ruby test_mock_rimesync.rb

    To run individual test¶ ↑

    $ ruby filename.rb -n TestName
